# PyConstant

New way to make constant in Python

## API Reference

```python
from pyconstantt import constant
```

#### Set Item

```python
  constant.Pi = 3.14
```

#### Get item

```python
  print(constant.Pi) # 3.14
```

## Installation

Install my-project with PyPl

```bash
  pip install pyconstantt
```
